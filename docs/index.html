Basics

Introduction

A high performance CSS-in-JS library for React that supports server-side rendering. Built similar to other popular libraries, with a familiar API, providing a great overall developer experience and a small footprint ~4.5kb gzipped

JavaScript Object Notation syntax is the chosen way to declare styles, meaning instead of writing properties in kebab-case like regular CSS, they are in camelCase eg: max-width would be maxWidth. Don't worry, all CSS properties are fully typed (thanks to CSSType) and provides autocompletion.

There is also a way to define global styles using CSS, more on that later.

Installation

Install Styled using your favourite package manager.

# with npm
npm install [package-name]

# with yarn
yarn add [package-name]

Don't forget to install react if you haven't already.

Once installed, just import like so

import styled from '[package-name]'

Getting started

Let's start by building a simple button.

const buttonAccent = 'hotpink';

const Button = styled.button({
  appearance: 'none',
  backgroundColor: buttonAccent,
  border: `2px solid ${buttonAccent}`,
  borderRadius: '4px',
  fontSize: '0.875rem',
  lineHeight: '1.5rem',
  margin: 0,
  padding: '0 6px',
  transition: '0.2s linear'
});

const Composition = () => (
  <Button>
    I'm a button
  </Button>
);

Pseudo

Styled comes with a helper (see API section for full details) for writing pseudo styles, below is an example of pseudo-class styles.

import { style } from '[package-name]'

const Button = styled.button({
  // base styles

  [style.hover]: {
    backgroundColor: 'rgba(255, 105, 180, 0.7)'
  },

  // you can write it yourself, if preferred
  ':hover': {
    backgroundColor: 'rgba(255, 105, 180, 0.7)'
  }
});

const Composition = () => (
  <Button>
    I'm a button
  </Button>
);

Similarly pseudo-element styles are defined as shown below.

import { style } from '[package-name]'

const Icon = styled.i({
  display: 'inline-block',
  position: 'relative',
  marginLeft: '4px',
  width: '24px',
  height: '24px',

  [style.before]: {
    content: '',
    position: 'absolute',
    top: '50%',
    left: '50%',
    borderTop: '6px solid transparent',
    borderBottom: '6px solid transparent',
    borderLeft: '6px solid white',
    width: 0,
    height: 0,
    transform: 'translate(-50%, -50%)'
  },

  // the same can be done manually
  '::before': {
    content: '',
    position: 'absolute',
    top: '50%',
    left: '50%',
    borderTop: '6px solid transparent',
    borderBottom: '6px solid transparent',
    borderLeft: '6px solid white',
    width: 0,
    height: 0,
    transform: 'translate(-50%, -50%)'
  }
});

const Composition = () => <Icon />;

Attribute selectors

You can event create selectors based on attributes.

const Link = styled.a({
  // Button base styles

  '[href]': {
    textDecoration: 'none'
  }
});

const Composition = () => (
  <Link href='#'>
    I'm a Link
  </Link>
);
